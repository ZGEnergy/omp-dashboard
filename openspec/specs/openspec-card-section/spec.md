## Removed

This capability was removed by the `openspec-folder-card-ui` change.

The OpenSpec section no longer lives inside the session card accordion. Functionality was split into:

- **`openspec-folder-section`** — Folder-level change list, artifact letters, refresh, bulk archive
- **`openspec-attach-combo`** — Session-level attach combo box, action buttons, detach

See those capabilities for current requirements.
